# 📰 Fake News Detection System using NLP

## 📌 About the Project
This project focuses on detecting whether a news article is **real or fake** using Natural Language Processing (NLP) and Machine Learning techniques. With the increasing spread of misinformation online, this system helps in identifying unreliable or misleading news content.

## 🎯 Objective
The main objective is to build a model that can automatically classify news articles into:
- ✅ Real News  
- ❌ Fake News  

## ⚙️ How It Works
1. Collect dataset (WELFake Dataset)  
2. Clean and preprocess text data  
3. Convert text into numerical form using **TF-IDF**  
4. Train model using **Passive Aggressive Classifier**  
5. Predict whether news is real or fake  

## 🛠️ Technologies Used
- Python  
- Scikit-learn  
- Pandas, NumPy  
- Flask (Backend)  
- HTML, CSS, JavaScript (Frontend)  

## 📊 Model Formula
TF-IDF:

$$
TF\text{-}IDF = TF(t,d) \times IDF(t)
$$

## 📈 Results
- Achieved high accuracy in classification  
- Evaluated using confusion matrix
